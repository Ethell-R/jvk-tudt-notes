  1. What is responsive design?

    What is responsive design?

    • It is an approach to designing Web sites, which provides the best possible user experience regardless of the device used to access the designed site.
    • It is not a technology, but is implemented with the use of various technologies and techniques:
      • HTML is fluid and goes some way to being responsive
      • Responsive layout technologies: flexbox, grid, columns (float was used in the past)
      • Responsive typography and media
      • Media queries
